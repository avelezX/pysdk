# PySdk - Xerenity Python SDK

## Qué es
SDK en Python para interactuar con la plataforma Xerenity. Proporciona funciones y utilidades para acceder a datos financieros, series de tiempo, y cálculos de la plataforma.

## Stack
- Python 3

## Comandos
```bash
pip install -r requirements.txt
```

## Reglas
- Siempre referenciar una issue en commits: `closes #X`
- No pushear directo a main, usar PRs
- Funciones con type hints y docstrings
- snake_case para funciones y variables
